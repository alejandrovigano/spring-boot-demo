# DEMO

Aplicacion basada en spring boot + data jpa + hateoas + h2-db + maven

## Endpoints


Buscar Personas
```
 GET - /demo/all
``` 

Buscar Persona
```
  GET - /demo/{id}
``` 

Crear propietario
```
  POST - /demo
``` 

Borrar persona
```
  DELETE - /demo/{id}
``` 
  
Actualizar persona
```
  PUT -	/demo/{id}
```
